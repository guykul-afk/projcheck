# ProjectCheck - מערכת בדיקת התכנות נדל"ן

מערכת מתקדמת לניתוח התכנות פיננסית של פרויקטי נדל"ן, הכוללת ניהול תקציב, ניהול מלאי דירות, ניתוח ריווחיות וחיזוי תזרים מזומנים מפורט.

## 🚀 קישורים מהירים
- **גישה מקומית (בזמן פיתוח):** [http://localhost:5600/](http://localhost:5600/)
- **מאגר קוד ב-GitHub:** [https://github.com/guykul-afk/projcheck](https://github.com/guykul-afk/projcheck)

## ✨ יכולות עיקריות
- **ניהול תקציב**: מעקב אחר עלויות הקמה, מימון, אגרות והיטלים.
- **ניהול מלאי**: ניהול דירות יזם ובעלים, שטחים ומחירים.
- **ניתוח ריווחיות**: מטריצת רגישות לשינויים בעלויות ובתקבולים.
- **תזרים מזומנים**: חיזוי תזרים חודשי ומצטבר עם תנאי תשלום מתקדמים (20/80 ופריסה לינארית).

## 🛠️ התקנה והרצה
1. שכפול המאגר:
   ```bash
   git clone https://github.com/guykul-afk/projcheck.git
   ```
2. התקנת תלויות:
   ```bash
   npm install
   ```
3. הרצה בסביבת פיתוח:
   ```bash
   npm run dev
   ```
